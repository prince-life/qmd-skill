Quick Markdown Search — powered by qmd

What is qmd?

qmd is a local hybrid search engine for Markdown notes, docs, and knowledge bases. Index once, search fast — using keyword (BM25) or semantic vector search.

Install

Via Ruby Gem

gem install qmd

Via Bun

bun install -g https://github.com/tobi/qmd

Quick Start

qmd collection add /path/to/notes --name notes --mask "**/*.md"
qmd embed
qmd search "your query"

Common Commands

Command Description
qmd search "query"Fast keyword search (default)
qmd vsearch "query"Semantic similarity search
qmd get "path/to/file.md"Retrieve a full document
qmd updateRe-index changed files
qmd statusCheck index health
